Earthquakes
===========



# necessary preparations

* download data from the web

* edit file paths in data_modification.sh
* run data_modification.sh to modify downloaded data to have the desired format

* edit 0_parameters.R -- coordinates etc.

* run Rscript main.R

* admire the map


# acknowledgements
